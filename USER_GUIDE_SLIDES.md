## 슬라이드 1: 돌콩봇 소개

### 🤖 돌콩봇이란?

**AI 기반 문서 검색 및 질의응답 시스템**

- 📚 회사 문서를 학습하는 똑똑한 AI
- 💬 자연어로 질문하면 정확한 답변 제공
- 🔍 의미 기반 문서 검색
- ⚡ 빠르고 정확한 정보 검색

### 핵심 가치
> "언제 어디서나, 필요한 정보를 바로 찾아주는 나만의 AI 비서"

## 슬라이드 5: RAG 채팅 사용법

### 🤖 채팅으로 정보 찾기

#### 사용 방법
1. **"🤖 RAG 채팅" 탭 클릭**
2. **질문 입력**
3. **전송 (Enter)**
4. **AI 답변 확인**

#### 효과적인 질문 예시

✅ **좋은 질문**
```
"연차 휴가 신청 방법을 알려주세요"
"3층 회의실 예약 방법이 궁금합니다"
"출장 경비 정산은 어떻게 하나요?"
```

❌ **나쁜 질문**
```
"휴가"  ← 너무 짧음
"이거"  ← 불명확함
```

## 슬라이드 6: 문서 검색 사용법

### 🔍 빠른 검색으로 정보 찾기

#### 검색 3단계
1. **"🔍 문서 검색" 탭 선택**
2. **검색어 입력**
3. **결과 확인**

#### 검색 팁
- 🎯 **의미 검색**: 정확한 단어가 아니어도 OK
  - "휴가 신청" → "연차 사용" 등도 검색됨
  
- 📝 **자연어 질문** 가능
  - "회의실은 어떻게 예약하나요?"

- ⭐ **유사도 점수**: 높을수록 관련성 높음

---

## 슬라이드 7: FAQ 활용

### 💬 자주 묻는 질문 바로 찾기

#### FAQ 접근 방법
1. **RAG 채팅 화면** 우측 하단
2. **"💬 FAQ" 아이콘** 클릭
3. **카테고리별** 질문 목록 확인

#### FAQ 카테고리
- 📋 **일반**: 시스템 사용법
- 💼 **인사**: 휴가, 근태, 급여
- 🏢 **업무**: 업무 프로세스
- 🔧 **기술**: 시스템 지원
- 📌 **기타**

#### 활용 팁
- 질문 클릭 → 채팅에 자동 입력
- FAQ에 없으면 → 이메일 문의

---

## 슬라이드 8: 이메일 문의

### 📧 직접 담당자에게 문의하기

#### 문의 방법
1. **FAQ 팝업** 열기
2. **"이메일 문의"** 버튼 클릭
3. **제목과 내용** 작성
4. **전송** 클릭

#### 효과적인 문의 작성
```
제목: [카테고리] 간단한 제목
예: [시스템 오류] 문서 업로드 실패

내용:
- 언제 발생했나요?
- 어떤 상황인가요?
- 오류 메시지는?
- 파일명/크기는?
```

## 슬라이드 9: 관리자 기능 - 문서 관리

### 📁 문서 업로드 및 관리

⚠️ **관리자 권한 필요**

#### 문서 업로드
1. **"⚙️ 관리자" 탭** 클릭
2. **"📁 문서 관리"** 선택
3. **파일 드래그 앤 드롭** 또는 선택
4. **업로드 완료 대기**

#### 지원 파일
- 📄 PDF (.pdf)
- 📝 Word (.docx)
- 📊 Excel (.xlsx)
- 📃 Text (.txt)

#### 제한 사항
- 최대 크기: **10MB**
- 처리 시간: 1MB당 **10~30초**

## 슬라이드 10: 관리자 기능 - FAQ 관리

### 💬 FAQ 등록 및 수정

#### FAQ 추가
1. **"💬 FAQ 관리" 탭** 선택
2. **"+ FAQ 추가"** 버튼
3. **정보 입력**
   - 카테고리
   - 질문
   - 답변
   - 키워드
4. **저장**

#### FAQ 수정/삭제
- ✏️ 수정: 항목 클릭 → 수정 버튼
- 🗑️ 삭제: 삭제 버튼 → 확인

## 슬라이드 11: 문제 해결 - 자주 발생하는 문제

### ❓ 자주 발생하는 문제와 해결법

| 문제 | 해결 방법 |
|------|-----------|
| 🚫 **로그인 안 됨** | 계정 확인, 캐시 삭제 |
| 💬 **답변 없음** | 문서 확인, 질문 구체화 |
| 🔍 **검색 안 됨** | 검색어 단순화, 다른 단어 시도 |
| 📤 **업로드 실패** | 파일 크기/형식 확인 |
| 🐌 **느린 속도** | 잠시 후 재시도, 캐시 삭제 |

### 📞 도움이 필요하면?
→ **이메일 문의** 또는 **관리자 연락**

## 슬라이드 22: Q&A

### ❓ 자주 묻는 질문 Top 5

#### Q1: 모든 직원이 사용할 수 있나요?
**A**: 네, 네이버웍스 계정이 있는 모든 직원이 사용 가능합니다.

#### Q2: 모바일에서도 되나요?
**A**: 네, 모바일 브라우저에서 접속 가능합니다.

#### Q3: 검색 기록이 남나요?
**A**: 시스템 로그는 관리 목적으로 보관되며, 개인정보는 보호됩니다.

#### Q4: 문서는 얼마나 업로드할 수 있나요?
**A**: 개별 파일은 10MB까지, 전체 용량은 서버 용량에 따릅니다.

#### Q5: 답변이 틀리면 어떻게 하나요?
**A**: 관리자에게 피드백 주시면 문서를 업데이트하여 개선합니다.
